# Installation instrustion

## Install Golang in Linux

    # cd /opt
    # wget https://dl.google.com/go/go1.13.4.linux-amd64.tar.gz
    # tar xf go1.13.4.linux-amd64.tar.gz 
    # rm -rf /usr/local/go
    # mv /opt/go /usr/local
