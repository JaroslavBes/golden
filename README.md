# Golden Point 1.x

Golden Point is Fido point package

## Compile

Golden Point use Task build system. See more https://taskfile.dev/

To compile:

    # task

## Usage guide

Golden Point open in you Web-browser on address
 
   http://127.0.0.1:8080/
